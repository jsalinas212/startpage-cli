# startpage-cli

## Description
I was bored, so I built a python script to fetch search results from Startpage.

## Requirements

### Python Version
* Tested using Python3.11

### Dependencies
* BeautifulSoup
* Requests
* argparse
* re

## Usage
Run the app.
```sh
python3 startpage-cli.py -q "<query>"
```

Ask for help.
```sh
python3 startpage-cli.py -h
```
